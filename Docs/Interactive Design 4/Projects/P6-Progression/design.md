# P6 — Progression
**Lectures:** L12–L13
**Status:** Not started

## Description
A space that goes somewhere — phases, arcs, escalation. Not just memory (the room changed because you were here) but directed transformation (the experience unfolds toward something). The rules themselves change. This is the most mature project: it uses all previous interaction vocabulary but sequences it over time. The room starts in one state, your actions unlock the next phase, visual language transforms, new inputs become available. Students learn to pace an experience.

## Interaction Core
- **Input:** All previous inputs, but gated and sequenced — what's available changes per phase
- **Variable relationships:** Sequential (variable advances through ordered phases), all previous types layered within phases
- **Output:** Full environment transformation across phases — lighting, atmosphere, geometry, sound, available interactions

## Interaction Palette
- Phase gating — IntVariable or state enum tracks current phase, conditions unlock new zones/abilities
- Rule transformation — interactions that work differently in phase 2 than phase 1
- Environmental arc — the entire visual/audio identity of the space shifts across phases
- Escalation — each phase raises stakes, density, or complexity of interaction
- SequenceRunner — ordered progression through authored milestones
