- Scale of tree grows uniformly over grow time

- I have 2 materials on the tree with two shaders 
-> bark material with Idyllic Fantasy Nature/ Surface shader -> variable Base Map Scale -> goes from 0 to 5

-> leaves shader -> Idyllic Fantasy Nature/ Vegetation -> Blend Height -> goes from .1 to .75

from same shader alpha cutoff from 1 to .2
